# Stacks Reward Points DApp

Minimal on-chain reward points tracker on Stacks blockchain.

## Repository Structure


## Contract

- `reward-points.clar` — reward points contract
- Public functions:
  - `add-reward(amount)` → add points to caller
  - `redeem(amount)` → redeem points from caller
  - `get-points(user)` → check points balance
